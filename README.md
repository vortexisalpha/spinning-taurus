# Spinning Torus

Small Three.js visualization that renders a wireframe torus and animates it in 3D with WebGL.

## What This Demonstrates

- Three.js scene setup.
- Perspective camera and WebGL renderer configuration.
- Mesh creation using `TorusGeometry` and `MeshStandardMaterial`.
- Basic point and ambient lighting.
- Frame-by-frame animation with `requestAnimationFrame`.

## Run Locally

```bash
npm install
npm run dev
```

If the project was created with an older bundler setup, use the script available in `package.json`.

## Main Files

```text
main.js      Three.js scene, torus mesh, lighting, and animation loop
style.css    Page/canvas styling
index.html   Canvas host page
```
